#  Bollywood Movie Recommendation System

A machine learning-based web application that recommends Bollywood movies based on similarity using a content-based filtering approach.

##  Features

-  Movie recommendations based on similarity
-  Uses real IMDB dataset
-  Fast and interactive UI with Streamlit
-  Content-based filtering system

##  Installation

git clone https://github.com/your-username/bollywood-movie-recommendation.git
cd bollywood-movie-recommendation
pip install -r requirements.txt
streamlit run src/app_streamlit.py

Then open your browser at:
http://localhost:8501
